## Welcome to Harsh

Harsh is a C precompiler written in C.

## Getting Started

From source code:

```
git clone git@github.com:vgsantoniazzi/harsh.git
```

#### Compile

```
make
```

#### Usage (REPL)

```
make run
```

#### Clean object files

```
make clean
```

#### Install

```
make install
```

#### Format source code

```
make format
```
