## Welcome to Harsh

Harsh is a C compiler written in C.

## Getting Started

From source code:

```
git clone git@github.com:vgsantoniazzi/harsh.git
cd harsh
clang lex.c -o lex
```

## Usage (REPL)

```
./lex
```
